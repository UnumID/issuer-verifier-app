## Description
A web app to interface with and showcase the Issuer and Verifier SDKs.

## Install

```sh
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
